# HR Analytics Dashboard

## Project Overview
This project is an interactive HR Analytics Dashboard developed using Power BI.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel

## Features
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Experience
- Department-wise Analysis
- Salary Slab Analysis
- Gender Analysis
